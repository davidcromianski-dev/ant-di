# Arquivos Públicos

## Ícone da Formiga

Para usar o ícone de formiga do Flaticon:

1. Baixe a imagem `@ant.png` do Flaticon
2. Renomeie para `ant.png`
3. Coloque neste diretório (`public/ant.png`)

O ícone será automaticamente usado no site.
